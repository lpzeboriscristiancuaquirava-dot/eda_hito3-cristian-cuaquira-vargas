# Proyecto EDA Hito 3

API REST desarrollada con Spring Boot para estructuras de datos.

## Tecnologías usadas
- Java 17
- Spring Boot
- Maven
- GitHub

## Funcionalidades

### Arreglo Unidimensional
- Agregar notas
- Mostrar notas
- Promedio
- Nota máxima
- Nota mínima

### Arreglo Bidimensional
- Gestión de asientos
- Ocupar asiento
- Liberar asiento
- Mostrar ocupados

### Lista Enlazada
- Agregar estudiantes
- Buscar estudiantes
- Ordenar por puntaje
